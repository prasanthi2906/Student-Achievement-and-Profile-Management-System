## ❗ COMMON ISSUES & SOLUTIONS

### Server Won't Start:
1. Kill existing Python processes
2. Check for port conflicts (netstat -ano | findstr :8000)
3. Run `python manage.py check` for diagnostics

### Admin Panel Errors:
1. Usually format_html syntax issues
2. Fixed in current version
3. All panels should work correctly

### Registration Not Working:
1. Check TemporaryRegistration admin
2. Verify admin approval process
3. Ensure student creation workflow
